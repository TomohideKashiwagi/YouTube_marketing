# YouTube流行りチェック

YouTubeで指定したキーワードに関連する動画を検索し、直近2週間で急激に再生回数が伸びている動画を一覧で見つけ出すシステムです。

## 機能

- キーワードでYouTube動画を検索
- 直近2週間で公開された動画を対象
- 急成長スコア（1日あたりの再生回数など）を計算
- 急成長している動画をスコア順に一覧表示

## セットアップ

### 1. リポジトリのクローン

```bash
git clone https://github.com/TomohideKashiwagi/YouTube_marketing.git
cd YouTube_marketing
```

### 2. 依存関係のインストール

```bash
pip install -r requirements.txt
```

### 3. YouTube Data API v3 のAPIキーを取得

1. [Google Cloud Console](https://console.cloud.google.com/) にアクセス
2. 新しいプロジェクトを作成（または既存のプロジェクトを選択）
3. 「APIとサービス」→「ライブラリ」から「YouTube Data API v3」を有効化
4. 「認証情報」→「認証情報を作成」→「APIキー」を選択してAPIキーを取得

### 4. 環境変数の設定

プロジェクトのルートディレクトリに `.env` ファイルを作成し、以下のように設定してください：

```
YOUTUBE_API_KEY=your_api_key_here
```

## 使用方法

### 方法1: ローカルで実行

```bash
python youtube_trending_finder.py
```

実行後、検索キーワードの入力を求められます。キーワードを入力すると、急成長している動画が一覧で表示されます。

### 方法2: Google Colabで実行（Googleドライブにデプロイ）

1. **Googleドライブにノートブックをアップロード**
   - `YouTube_Trending_Finder.ipynb`ファイルをGoogleドライブにアップロード
   - または、Googleドライブから直接ノートブックを開く

2. **Google Colabで開く**
   - ファイルを右クリック → 「アプリで開く」→「Google Colaboratory」を選択
   - または、[Google Colab](https://colab.research.google.com/)にアクセスして、ドライブからノートブックを開く

3. **実行**
   - セルを上から順番に実行（Shift + Enter）
   - APIキーは既に設定済みですが、必要に応じて変更してください
   - 最後のセルで検索キーワードを変更して実行

**メリット:**
- Googleドライブから直接アクセス可能
- 環境構築不要（Colab上で自動的にライブラリがインストールされる）
- ブラウザからどこでも実行可能

## 急成長スコアについて

急成長スコアは以下の要素で計算されます：

- 1日あたりの再生回数
- 総再生回数
- 公開からの経過日数（新しいほど高得点）

スコアが高いほど、急激に再生回数が伸びている動画と判断されます。

## 注意事項

- YouTube Data API v3には1日のリクエスト上限があります（無料プランでは10,000単位/日）
- APIキーは公開リポジトリにコミットしないでください（`.env`ファイルは`.gitignore`に含まれています）
